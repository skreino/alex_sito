# ALEX ROX Tattoo Website

Premium editorial website for ALEX ROX, a Chicano-style black and grey tattoo artist in Busnago MB.

## Stack

- Next.js
- React
- Tailwind CSS
- Framer Motion

## Commands

```bash
npm install
npm run dev
npm run build
```
